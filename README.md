🌐 Translator Web App

A modern web-based translation application built using HTML, CSS, and JavaScript, powered by the Microsoft Azure Translator API. This app allows users to translate text between multiple languages in real time with an interactive and user-friendly interface.

🚀 Features
🌍 Real-time text translation
🔄 Language swap functionality
🎯 Simple and clean UI/UX
⚡ Fast API integration
🔔 Popup notifications for user feedback
📱 Responsive design
🛠️ Tech Stack
HTML
CSS
JavaScript
Microsoft Azure Translator API
📂 Project Structure
project-folder/
│
├── index.html
├── styles.css
└── script.js
⚙️ Setup Instructions
Clone or download the project
Open the folder in VS Code
Replace the API key in script.js:
const apiKey = "YOUR_API_KEY_HERE";
Run using Live Server
Open in browser and start translating
🔐 Important Note

Do not expose your API key in production. For secure usage, move API calls to a backend server.
